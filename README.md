## Installation

```bash
$ npm install
```

#### Docker

There is a `docker-compose.yml` file for starting Docker only for database PostgreSQL.

`docker-compose up`

After running the sample, you can stop the Docker container with

`docker-compose down`


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
